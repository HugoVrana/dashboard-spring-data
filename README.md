### Dahsboard Application API

This is a sample application for learning Spring + next.js stack

The Application is a simulation of an company called Acme, tracking their invoices to customers

This project is based on a GitLab [Project Template](https://docs.gitlab.com/ee/gitlab-basics/create-project.html).

## Frontend
The frontend is a next.js with tailwind app hosted on [Vercel](https://nextjs-dashboard-two-beryl.vercel.app/).

Repository can he found [here](https://gitlab.com/hugo.vrana/nextjs-dashboard).

## API / Backend
This API is a spring API hosted on [Render] (https://spring-dashboard-1.onrender.com)

Repository can be found [here] (https://gitlab.com/hugo.vrana/spring-dashboard)
